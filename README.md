## 🚀 Overview

N'apsak?, kullanıcıların plan oluşturmasını, keşfetmesini ve sosyal olarak paylaşmasını sağlayan AI destekli bir mobil uygulamadır.

Kullanıcılar:
- Plan oluşturur
- AI’dan öneri alır
- Lokasyona göre keşif yapar
- Planlarını paylaşır

---

## 🧱 Tech Stack

**Frontend**
- React Native (Expo)

**Backend**
- FastAPI
- PostgreSQL

**AI**
- Local LLM (Ollama - Mistral / DeepSeek)

---

## 📱 Core Features (MVP)

- 🔐 Google ile giriş (JWT auth)
- 👤 Kullanıcı profili
- 🗺️ Lokasyon bazlı plan oluşturma
- 🤖 AI plan önerisi
- 📰 Sosyal feed (plan paylaşımı)
- ❤️ Like & 💬 Comment sistemi
- 📍 Mekan & etkinlik önerileri
